# Pravidla pro přispívání do projektu BarX

1. **Forkni repozitář** a vytvoř novou větev pro své změny.
2. **Popiš změny** v commitu jasně a výstižně.
3. **Před odesláním PR** spusť všechny testy (`npx playwright test`).
4. **Dodržuj strukturu a styl kódu** (viz `styles.css`, používej sémantické HTML, čistý JS).
5. **Přidávej komentáře** k nejasným částem kódu.
6. **Respektuj existující dokumentaci** a aktualizuj ji při změnách.
7. **V případě větších změn** nejprve otevři issue k diskuzi.
